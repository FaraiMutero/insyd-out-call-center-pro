# InsydOut Call Center Pro

Workspace initialized for a monorepo architecture:

- `server/` - Node.js + Express API
- `client/` - React + Vite SPA
- `data/` - local runtime data (SQLite DB + recordings)

## Quick start

1. Install dependencies:
   - `npm install`
2. Start API:
   - `npm run dev`
3. Start SPA (separate terminal):
   - `npm run dev:client`

## Build and run

- Build: `npm run build`
- Start production server: `npm start`
